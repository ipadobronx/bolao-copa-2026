## Task 2 — Criar `app/opengraph-image.tsx`

### Arquivo

`app/opengraph-image.tsx` (novo)

### Estrutura geral

```ts
import { ImageResponse } from 'next/og';

export const runtime = 'edge';
export const alt = 'Mala na Copa — Bolão da Copa do Mundo 2026 · R$ 10.000 em prêmios';
export const size = { width: 1200, height: 630 };
export const contentType = 'image/png';

const FLAGS_ISO = ['fr', 'es', 'gb-eng', 'br', 'ar', 'pt', 'de', 'nl', 'no', 'ch', 'be', 'co', 'uy'] as const;

async function loadBebasNeue(): Promise<ArrayBuffer | null> {
  try {
    const cssRes = await fetch(
      'https://fonts.googleapis.com/css2?family=Bebas+Neue&display=swap',
    );
    if (!cssRes.ok) return null;
    const css = await cssRes.text();
    const fontUrl = css.match(/url\((https:\/\/[^)]+\.(?:woff2|ttf))\)/)?.[1];
    if (!fontUrl) return null;
    const fontRes = await fetch(fontUrl);
    if (!fontRes.ok) return null;
    return await fontRes.arrayBuffer();
  } catch {
    return null;
  }
}

export default async function OpengraphImage() {
  const bebas = await loadBebasNeue();

  return new ImageResponse(
    (
      <div
        style={{
          width: '100%',
          height: '100%',
          display: 'flex',
          flexDirection: 'column',
          background:
            'radial-gradient(circle at 85% 15%, rgba(16,185,129,0.18) 0%, transparent 45%), linear-gradient(135deg, #0a0e1a 0%, #111827 100%)',
          padding: '60px 80px',
          fontFamily: bebas ? 'Bebas Neue' : 'sans-serif',
        }}
      >
        {/* Top row: wordmark + AO VIVO badge */}
        <div style={{ display: 'flex', justifyContent: 'space-between', alignItems: 'center' }}>
          <div style={{ color: '#f8fafc', fontSize: 64, letterSpacing: '0.04em', display: 'flex' }}>
            MALA na COPA
          </div>
          <div style={{ display: 'flex', alignItems: 'center', gap: 12 }}>
            <div style={{ width: 14, height: 14, borderRadius: 999, background: '#10b981' }} />
            <div
              style={{
                color: '#10b981',
                fontSize: 22,
                fontFamily: 'monospace',
                letterSpacing: '0.1em',
                display: 'flex',
              }}
            >
              RANKING AO VIVO
            </div>
          </div>
        </div>

        {/* Center: headline */}
        <div
          style={{
            display: 'flex',
            flexDirection: 'column',
            alignItems: 'center',
            justifyContent: 'center',
            flex: 1,
            gap: 20,
          }}
        >
          <div
            style={{
              color: '#facc15',
              fontSize: 180,
              lineHeight: 0.9,
              letterSpacing: '-0.02em',
              display: 'flex',
            }}
          >
            R$ 10.000
          </div>
          <div
            style={{
              color: '#f8fafc',
              fontSize: 64,
              letterSpacing: '0.04em',
              display: 'flex',
            }}
          >
            EM PRÊMIOS
          </div>
          <div
            style={{
              color: '#cbd5e1',
              fontSize: 30,
              fontFamily: 'sans-serif',
              marginTop: 20,
              display: 'flex',
            }}
          >
            Bolão da Copa 2026 · PIX · Top 3 leva tudo
          </div>
        </div>

        {/* Bottom: flags row + footer URL */}
        <div style={{ display: 'flex', flexDirection: 'column', gap: 24 }}>
          <div style={{ display: 'flex', gap: 14, justifyContent: 'center' }}>
            {FLAGS_ISO.map((iso) => (
              <img
                key={iso}
                src={`https://flagcdn.com/w40/${iso}.png`}
                width={48}
                height={32}
                style={{ borderRadius: 4 }}
                alt=""
              />
            ))}
          </div>
          <div
            style={{
              display: 'flex',
              justifyContent: 'flex-end',
              color: '#94a3b8',
              fontFamily: 'monospace',
              fontSize: 24,
            }}
          >
            malanacopa.com.br
          </div>
        </div>
      </div>
    ),
    {
      ...size,
      fonts: bebas
        ? [{ name: 'Bebas Neue', data: bebas, style: 'normal', weight: 400 }]
        : [],
    },
  );
}
```

### Atenção técnica

- **Todo elemento que tem mais de 1 filho precisa de `display: 'flex'`** — restrição do Satori (engine por trás do `next/og`). Mesmo `<div>` com texto puro mais um irmão exige flex.
- `fontFamily: bebas ? 'Bebas Neue' : 'sans-serif'` — fallback automático.
- `<img>` em ImageResponse aceita URLs externas. Headers padrão do flagcdn permitem hot-linking.
- `gb-eng` é suportado pelo flagcdn (verificar com curl antes se quiser). Se falhar, troca pra `gb` (Union Jack).
- Edge runtime: sem `node:`, sem `process.env` exceto via `process.env.NEXT_PUBLIC_*`. Não precisa de env aqui.

### Validação manual

Após criar o arquivo:

1. `pnpm dev` → abrir `http://localhost:3000/opengraph-image` no browser.
2. Imagem deve aparecer renderizada — Bebas Neue se Google Fonts respondeu, sans-serif caso contrário.
3. Texto preservado em ambos os casos.

## Task 5 — Criar `app/robots.ts` e `app/sitemap.ts`

### `app/robots.ts`

```ts
import type { MetadataRoute } from 'next';

export default function robots(): MetadataRoute.Robots {
  return {
    rules: [{ userAgent: '*', allow: '/', disallow: ['/admin', '/api'] }],
    sitemap: 'https://malanacopa.com.br/sitemap.xml',
  };
}
```

### `app/sitemap.ts`

```ts
import type { MetadataRoute } from 'next';

export default function sitemap(): MetadataRoute.Sitemap {
  const base = 'https://malanacopa.com.br';
  const lastModified = new Date();
  return [
    { url: base, lastModified, priority: 1, changeFrequency: 'weekly' },
    { url: `${base}/login`, lastModified, priority: 0.8, changeFrequency: 'monthly' },
    { url: `${base}/termos`, lastModified, priority: 0.5, changeFrequency: 'yearly' },
    { url: `${base}/privacidade`, lastModified, priority: 0.5, changeFrequency: 'yearly' },
  ];
}
```

### Atenção

- Path: arquivos em `app/` (não `app/(public)/`). Next reconhece `robots.ts` e `sitemap.ts` no root do `app/` como rotas especiais.
- `lastModified` usa `new Date()` no momento do request (cada deploy reseta).
- `changeFrequency` é hint pro crawler, não obrigatório — adicionei pra clarear semântica.

## Task 6 — Validação local

### Comandos

```bash
pnpm typecheck
pnpm lint
pnpm build
```

### Critérios

- `typecheck` sem erros.
- `lint` sem erros novos (warnings preexistentes em outros arquivos OK).
- `build` completa com sucesso. Saída deve incluir:
  - `/opengraph-image` (Edge)
  - `/icon` (Edge)
  - `/apple-icon` (Edge)
  - `/robots.txt` (Static)
  - `/sitemap.xml` (Static)

### Possíveis erros e fix

| Erro | Causa | Fix |
|------|-------|-----|
| `Module not found: next/og` | Next < 13.3 (improvável aqui — já estamos em 14.2.18) | n/a |
| `display: flex` warning do Satori | Faltou `display: 'flex'` em algum elemento com 2+ filhos | Adicionar à mão. |
| `Network error` em font fetch durante build | Build não tem rede (improvável no Vercel) | Try/catch já trata. |
| Image size > 1MB | Muitas flags ou font enorme | Reduzir flags pra 5–6 (apenas favoritas) ou reduzir size. |

### Commit

Não há commit dedicado — validação só. Se algum fix necessário, commit `fix(F18): <razão>`.
